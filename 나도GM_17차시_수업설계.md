## 5. 핵심 데이터 지표 사전

### 투수 지표

| 지표 | 풀네임 | 중학생 설명 | 데이터 출처 |
|------|--------|-------------|-------------|
| **IVB** | Induced Vertical Break | 중력을 거스르고 공이 '버티는 힘'. 클수록 타자 눈에 공이 떠오르는 것처럼 보임 | MLB Savant |
| **VAA** | Vertical Approach Angle | 홈플레이트를 통과할 때 지면과 이루는 각도. **절댓값이 작을수록**(0도에 가까울수록) 타자가 치기 어려움 | MLB Savant |
| **FIP** | Fielding Independent Pitching | 수비를 배제한 순수 투수 실력. ERA보다 실력을 더 정확히 반영 | Statiz |
| **BABIP** | Batting Average on Balls In Play | 인플레이 타구의 안타 비율. 높으면 불운, 낮으면 행운 → "저평가 투수 발굴 도구" | Statiz |
| **CSW%** | Called Strike + Whiff % | 타자가 멍하니 당하거나 헛스윙한 비율. 투수가 타자를 '요리'한 점수 | Statiz |
| **Putaway%** | - | 2스트라이크 이후 삼진으로 끝낼 수 있는 결정구의 위력 | Statiz |

### 타자 지표

| 지표 | 풀네임 | 중학생 설명 | 데이터 출처 |
|------|--------|-------------|-------------|
| **Exit Velocity** | 타구 속도 | 배트에 맞은 직후 공의 속도. 빠를수록 안타/홈런 확률↑ | MLB Savant |
| **Launch Angle** | 발사각 | 타구의 수직 각도. **25~35도**가 홈런/장타 최적 구간 | MLB Savant |
| **Barrel%** | 배럴 타구 비율 | 발사각 + 타구속도의 교집합에서 나온 '최고급 타구' 비율 | MLB Savant / Statiz |
| **wRC+** | Weighted Runs Created+ | 구장 효과와 리그 평균을 보정한 타자 생산성. **100이 리그 평균**, 클수록 우수 | Statiz |
| **WAR** | Wins Above Replacement | 이 선수가 없었다면 팀이 잃었을 승수. 투수·타자 통합 비교 가능 | Statiz |
| **OPS** | OBP + SLG | 출루율 + 장타율의 합산. 간단하면서도 득점 생산력과 상관관계 높음 | Statiz |

### VAA 핵심 포인트 (쉬운 설명)

```
VAA = -6도  →  위에서 아래로 가파르게 내리꽂힘 (땅볼 유도형)
VAA = -4도  →  지면과 거의 수평 (하이 패스트볼 특화, 삼진 유도형)

💡 투수 유형별 전략:
  - 삼진 잡는 투수 원함 → VAA 절댓값이 작은 투수 (평평한 진입각)
  - 땅볼 유도 원함     → VAA 절댓값이 큰 투수 (내리꽂는 진입각)
```

## 6. 나도GM 알고리즘

### 투수 능력치 계산식

```python
def calc_pitcher_power(pitcher):
    """
    투수의 종합 구위 점수 계산 (0~100점)
    - IVB: 중력 대비 수직 무브먼트 (cm 단위)
    - VAA: 수직 진입각 (음수, -4.0에 가까울수록 유리)
    - CSW%: 헛스윙+루킹 비율
    - FIP: 수비 무관 자책점 (낮을수록 좋음)
    """
    ivb_score  = min(pitcher['IVB'] / 50, 1.0) * 40     # 최대 40점
    vaa_score  = max(0, (5.0 - abs(pitcher['VAA'])) / 5) * 30  # 최대 30점
    csw_score  = min(pitcher['CSW%'] / 40, 1.0) * 20    # 최대 20점
    fip_score  = max(0, (6.0 - pitcher['FIP']) / 6) * 10 # 최대 10점
    return ivb_score + vaa_score + csw_score + fip_score

def calc_batter_power(batter):
    """
    타자의 종합 생산성 점수 계산 (0~100점)
    """
    wrc_score    = min(batter['wRC+'] / 160, 1.0) * 50
    barrel_score = min(batter['Barrel%'] / 20, 1.0) * 30
    ops_score    = min(batter['OPS'] / 1.0, 1.0) * 20
    return wrc_score + barrel_score + ops_score
```

### 시즌 시뮬레이션 로직

```python
import random

def simulate_season(roster, games=36, cumulative_wins=0, cumulative_losses=0):
    """
    roster: 전체 29명 선수 데이터 리스트
    games: 이번 분기 경기 수 (기본 36경기)
    """
    # 팀 전체 투수/타자 능력치 집계
    pitchers = [p for p in roster if p['position'] == 'P']
    batters  = [b for b in roster if b['position'] != 'P']

    team_pitch = sum(calc_pitcher_power(p) for p in pitchers) / len(pitchers)
    team_bat   = sum(calc_batter_power(b) for b in batters) / len(batters)

    # 리그 평균 대비 상대적 강도 → 승률 산출
    league_avg = 50  # 리그 평균 능력치
    win_rate = 0.5 + (team_pitch - league_avg) * 0.004 \
                   + (team_bat   - league_avg) * 0.004
    win_rate = max(0.2, min(0.85, win_rate))  # 승률 상한/하한 제한

    # 36경기 몬테카를로 시뮬레이션
    wins = sum(1 for _ in range(games) if random.random() < win_rate)

    return {
        'wins':   cumulative_wins + wins,
        'losses': cumulative_losses + (games - wins),
        'win_rate': (cumulative_wins + wins) / (cumulative_wins + cumulative_losses + games)
    }
```

### 포스트시즌 모드

```python
def playoff_series(team_a, team_b, series_type='3'):
    """
    series_type: '3' = 3판 2선승 (PO), '5' = 5판 3선승 (KS)
    단기전 보정: 에이스 가중치 1.5배, 불펜 투입 조기화
    """
    target_wins = {'3': 2, '5': 3}[series_type]
    wins_a, wins_b = 0, 0

    # 에이스 선발 강화 (단기전 모드)
    a_power = team_a['ace_power'] * 1.3
    b_power = team_b['ace_power'] * 1.3

    while wins_a < target_wins and wins_b < target_wins:
        if random.random() < a_power / (a_power + b_power):
            wins_a += 1
        else:
            wins_b += 1

    winner = 'A' if wins_a == target_wins else 'B'
    return winner, wins_a, wins_b
```

#### 🌀 5차시: [과학/물리] 궤적의 마법 — VAA & IVB

**학습 목표:** 마그누스 효과를 이해하고 투구 궤적 데이터(VAA, IVB)를 분석한다.

**핵심 과학 개념:**

> **마그누스 효과(Magnus Effect)**  
> 회전하는 물체 주위로 공기의 흐름이 비대칭이 되어 압력 차이가 발생, 공이 진행 방향과 수직으로 힘을 받는 현상

**IVB 실험:**
- 종이를 입술 아래 대고 불면 위로 뜨는 현상 관찰 → "공기가 빠르게 흐르는 쪽의 압력이 낮다"
- 이것이 야구공에서 회전에 의해 발생할 때가 IVB!

**VAA 시각화:**
- 높은 릴리스 포인트 → 가파른 진입각 (절댓값 큼, 땅볼 유도)
- 낮은 릴리스 포인트 → 평평한 진입각 (절댓값 작음, 삼진 유도)

**Orange3 실습:**

```
→ 관찰: "IVB가 높고 VAA 절댓값이 작은 구간에 헛스윙률이 높은가?"
```

**탐구 질문:**
1. IVB가 높은 투수와 낮은 투수의 헛스윙률(Whiff%)은 어떻게 다른가?
2. VAA의 절댓값이 작은 투수들의 탈삼진율은 어떤 경향을 보이는가?

#### 💎 6차시: [정보/수학] 저평가 에이스 발굴 — FIP & BABIP

**학습 목표:** 운과 실력을 구분하는 지표를 이해하고 저평가된 투수를 발굴한다.

**핵심 개념:**

```
ERA(자책점)    = 실제 결과 (수비 실수, 운 모두 포함)
FIP(수비무관)  = 순수 투수 능력 (삼진, 볼넷, 홈런만 반영)
BABIP          = 인플레이 타구 안타율
  → 리그 평균 BABIP ≈ 0.295~0.310
  → 높은 BABIP = 불운한 투수 (안타가 될 확률이 비정상적으로 높음)
  → 낮은 BABIP = 행운이 따른 투수
```

**보물찾기 활동:**
```
[Statiz 투수 데이터 로드 → Orange3]
X축: FIP  Y축: ERA
컬러: BABIP

→ "FIP는 낮은데(실력은 좋은데) ERA가 높은 점을 찾아라!"
→ "그 점의 BABIP는 높지 않은가?"
→ "이 투수는 불운에 빠진 숨겨진 에이스다!"
```
